IActionsContainer class
#include <mcore/uielements/iactionscontainer.h>

Container interface for QMenu and QActions classes.

Contents

  • Reference
    • Derived classes
    • Constructors, destructors, conversion operators
    • Public functions
    • Public slots

All items (QMenuBar, QMenu, QToolBar(not implemented)) that have menus are wrapped in this container implementing that interface. Containers can be nested. The container does not own the wrappable items. If they are destroyed, the corresponding slots are invoked and the item is removed from the container.
