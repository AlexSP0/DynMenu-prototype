UiQMenuBarElement class
#include <mcore/uielements/uiqmenubarelement.h>

The UiQMenuBarElement class container for QMenu object.

Contents

  • Reference

Container for an object of type QMenuBar. The container does not own the object. The container is deleted when the QMenu is destroyed.
